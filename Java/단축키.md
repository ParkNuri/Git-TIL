# 단축키

| ctrl + /                     | 한 줄 주석                                |
| ---------------------------- | ----------------------------------------- |
| ctrl + shift + / + 범위 설정 | 여러 줄 주석                              |
| ctrl + d                     | 여러줄 주석 해제                          |
| ctrl + shift + \ + 범위 설정 | 여러줄 주석 해제                          |
| ctrl + shift + o             | 한 번에 import / 사용하지 않는 imort 삭제 |
| ctrl + shift + f             | 들여쓰기 정리                             |
